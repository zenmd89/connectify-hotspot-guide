## 什么是 Connectify Hotspot？

**Connectify Hotspot 是一款 Windows 平台的虚拟路由器软件**，可以将电脑的有线网络、4G/5G 网卡或 VPN 连接，一键共享为 Wi-Fi 热点，让周围的手机、平板、游戏主机等设备直接联网。

根据官网数据，Connectify Hotspot 是全球用户数量最多的虚拟路由软件，累计用户超过 1.5 亿。它支持 Windows 10 和 Windows 11，有免费版、Pro 专业版、Max 旗舰版三个版本，付费版均为一次性买断，不需要按年订阅。

## 电脑变WiFi热点：为什么不直接用 Windows 自带功能？

说实话，Windows 10/11 自带的移动热点功能能解决大部分日常需求。问题在于它挑剔。

几个常见的崩溃场景：

- 笔记本插了 4G USB 网卡，系统热点无法识别这个网络来源
- 连了企业 VPN，想把 VPN 保护的网络共享给手机，系统热点直接不支持
- 台式机没有无线网卡，系统热点根本启动不了
- 某些老款无线网卡驱动版本，系统热点开了又自动关

Connectify Hotspot 的兼容性在这些场景里要强得多。它基本上能接受电脑能上网的任何方式，然后把网络共享出去。另外它还有一个 Wi-Fi 中继模式（Max 版独占），可以把电脑变成信号放大器，把家里死角的 Wi-Fi 信号扩展过去——这个功能是系统自带功能完全没有的。

## 如何设置电脑变WiFi热点：Connectify 使用步骤

拿到正版激活授权后，整个流程分两个阶段：激活 + 开热点。

### 激活许可证

1. 下载并安装 Connectify Hotspot（可在数码荔枝页面找到下载链接）
2. 打开软件，在顶部菜单选择「工具 > 激活许可证」
3. 弹出窗口中点击「已购买」
4. 在登录页面点击「忘记密码？」
5. 输入购买时使用的邮箱，点击「提交」
6. 收到开发商发来的邮件，里面包含临时密码
7. 返回登录页面，输入邮箱和密码，点击「登录」
8. 点击「激活」，完成

### 开启 WiFi 热点

1. 打开 Connectify 软件主界面
2. 在「热点名称（SSID）」栏填入你想要的 WiFi 名字（Pro/Max 版支持 emoji 和中文）
3. 设置热点密码（至少 8 位）
4. 在「共享网络来源」下拉菜单中选择你当前使用的网络接口
5. 点击「Start Hotspot」
6. 其他设备搜索到你设定的 WiFi 名字，输入密码连接即可

就这样。配置完一次之后，下次开机软件会自动记住设置，不用每次重新填。

**场景一：出差住酒店**。酒店 WiFi 按设备收费，一台设备一天 ¥30。用 Connectify 把笔记本连上酒店 WiFi 再开热点，手机、平板都往热点上连，只算一台设备的钱。Max 版还自带广告屏蔽，连进来的设备上广告也少很多。

**场景四：游戏主机入网（Max）**。老款游戏主机只有有线接口，用 Max 的桥接模式，把主机接到笔记本的网线口，再用 Connectify 把主机桥接进家里的局域网，局域网游戏、串流内容都能跑通。

## 常见问题解答

**Q：Connectify Hotspot 支持 Windows 7 吗？**
A：不支持。Connectify Hotspot 目前仅支持 Windows 10 及以上版本，Windows 7 和 8 均无法使用。

**Q：买 Pro 还是 Max？**
A：如果主要需求是把有线/4G 网络共享给其他设备，Pro 完全够。如果需要 Wi-Fi 中继（扩展信号范围）、桥接模式（游戏主机接入局域网）或自定义 IP/DHCP，选 Max。

**Q：换电脑之后授权还能用吗？**
A：可以。Connectify 支持授权转移，方式是先在旧设备上卸载软件，然后在新设备上重新安装并登录账号激活即可。
